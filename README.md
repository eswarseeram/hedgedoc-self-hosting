# HedgeDoc Self Hosting using Docker

## Overview
This project demonstrates how to self-host the HedgeDoc collaborative Markdown editor using Docker. HedgeDoc allows multiple users to collaboratively create and edit markdown documents in real time.

## Technologies Used
- Docker
- Docker Compose
- HedgeDoc
- SQLite Database

## Features
- Self-hosted collaborative markdown editor
- Containerized deployment using Docker
- Lightweight SQLite database
- Simple configuration using Docker Compose

## Setup Instructions

1. Install Docker and Docker Compose.

2. Create a docker-compose.yml file with the HedgeDoc configuration.

3. Run the container using:

docker-compose up -d

4. Open the browser and access:

http://localhost:3000

## Applications
- Collaborative note taking
- Team documentation
- Markdown based knowledge sharing
